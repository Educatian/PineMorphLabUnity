# PineMorph Lab Learning Guide

## 10-15 minute sequence

| Time | Learner action | Evidence focus |
| --- | --- | --- |
| 0-2 min | Complete the guided learning overlay | Active-passive structure and differential swelling |
| 2-4 min | Predict and test the aligned-fiber baseline | Under-opening and low mismatch strain |
| 4-6 min | Change fiber orientation | Transformed hygroexpansion and curvature |
| 6-8 min | Increase active-layer thickness | Curvature versus square-law diffusion time |
| 8-10 min | Create stiffness mismatch | Equilibrium redistribution and stress |
| 10-12 min | Optimize Trial 5 | Satisfy angle, time, and stress together |
| 12-15 min | Complete CER check and export CSV | Design recommendation with values, units, and limits |

## Common misconceptions

| Misconception | Productive check |
| --- | --- |
| A thicker active layer always improves motion | Compare opening angle with t95 response time. |
| Humidity supplies motor force | The actuator stores differential free strain; it has no motor. |
| Maximum opening is the best design | Inspect the useful angle band and stress limit. |
| Fiber direction changes only appearance | Follow the transformed hygroexpansion term. |
| One metric is enough | A viable design must satisfy all three constraints. |

## Claim-Evidence-Reasoning prompt

- **Claim:** Recommend a passive bilayer configuration for a humidity-responsive vent.
- **Evidence:** Cite opening angle, t95, and peak stress with units and limits.
- **Reasoning:** Connect fiber angle and layer properties to mismatch strain, laminate equilibrium, curvature, diffusion, and stress.

## Instructor facilitation

- Require a prediction before allowing discussion of the expected outcome.
- During Trials 2-4, ask what was held constant and which model term changed.
- During Trial 5, point only to the three limits and do not provide slider values.
- Distinguish calculated trial evidence from physical sensor measurements.
- End by asking which parameter would need physical calibration first.
